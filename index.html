    第一步

    快速下载

    本页面顶部提供了源代码的zip格式的压缩包和tar.gz格式的压缩包，请点击直接下载

    访问源代码页面

    源代码的图形页面访问地址是：https://github.com/feuyeux/jsfrf

    使用git下载源代码

    $ cd your_repo_root/repo_name
$ git clone git://github.com/feuyeux/jsfrf.git

    最后一步

    源代码使用Maven管理。 使用Eclipse开发： 第一次使用时，在mac/linux平台开发，请执行buildEclipse.sh，在windows平台使用请执行buildEclipse.bat，然后导入到IDE中。 使用Netbean，IntelliJ IDEA可以直接导入。
